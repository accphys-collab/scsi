scsi

Self-Consistent Symplectic Integrator for charged particle beam dynamics;
based on TPSA (Truncated Power Series Algebra),
aka PTC (Polymorphic Tracking Code).


Author: Johan Bengtsson

Requirements:

  1. GNU autoconf and automake environment.

  2. GNU C++ and GFORTRAN compiler: gcc and gfortran

To install:

  1. tar -xvzf <tar ball name> 

  2. cd scsi 

  3. ./bootstrap

  4. ./configure

  5. make install
